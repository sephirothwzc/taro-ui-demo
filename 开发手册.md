# 问题答疑

- 跨域问题
  - h5跨域 配置config>dev.js
  - request 参数 mode: 'no-cors',

```js
h5: {
  devServer: {
    proxy: {
      '/app-web': {
        target: 'http://10.199.209.111:8081',
        // pathRewrite: {
        //   '^/app-web': '/app-web',
        // },
        changeOrigin: true,
      },
    },
  },
},
```
